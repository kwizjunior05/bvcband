# Brazos Valley Community Band Website

This repository serves as a proposed replacement for the current BVCB Website. At its core, the site is designed to be:

1. Mobile and desktop friendly
2. Sleek, modern, and aesthetically appealing
3. Easy to maintain

For maintianability reasons, the website should be static. I want to minimize external dependencies and cross-origin requests.

I settled on HTML, CSS, JavaScript for several reasons:

1. Ubiquity. The more code I can generate with AI, the less work I need to do myself. There's plenty of examples (good and bad) for AI to learn from, so I figured this would be a good experiment for it.
2. Education. I wanted to learn what it's like to build a simple website with nothing but the basics. With my experience in Svelte, it's surprising I haven't actually done this yet. I might refactor it later if it becomes too much.
3. No dev environment needed. Edit the text in Notepad and click the HTML. Doesn't get easier than that.
4. Because I felt like it.
